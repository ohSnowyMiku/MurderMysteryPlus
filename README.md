# MurderMysteryPlus

![Powered by OneConfig](https://polyfrost.org/media/branding/badges/badge_1.svg)
![Minimal Powered by OneConfig](https://polyfrost.org/media/branding/badges/badge_3.svg)

This is a Minecraft 1.8.9 Forge client mod developed with the help of AI, designed to enhance your experience when playing Hypixel MurderMystery. Currently supports Simplified Chinese and English languages on Hypixel.

Note:
Use this mod at your own risk

## Features

### General

- **Hide No Kill Alert Title** - Hide the title alert when the killer has no kills within 1 minute
- **Show Player Roles in Tab** - Display player roles in the tab list for known players
- **Hide Useless Armor Stand Tags** - Hide name tags of invisible armor stands on the map
- **Hide Particles** - Hide most particles in the game to improve performance
- **Toggle Particles** - Keybind (default: H) to toggle particle visibility
- **Show Bow Drop Particles** - Show particles at the location where the bow drops

### Chat

- **Hide Kali Spam Message** - Hide Kali spam messages in chat
- **Auto Swap Chat Channel** - Automatically swap chat channel when joining or leaving a party
- **Auto Send Bow Kill Distance** - Automatically send bow kill distance messages
  - **Language** - Select language for distance messages (English/Chinese)
  - **Send to Public/Party/Guild** - Choose where to send distance messages
- **Auto Send Role to Party** - Automatically send your role information to party chat
  - **Language** - Select language for role messages (English/Chinese)
- **Party Quick Command Buttons** - Show quick command buttons for party management

### Item

- **Disable Gestures ArmorStand** - Disable gesture armor stands (usable with right-click while holding shift)
- **Protect Detective Bow** - Prevent the detective bow from being sacrificed to Kali (can still sacrifice while holding shift)
- **Protect Murderer Knife** - Prevent knives (e.g., boats) from disappearing due to vanilla mechanics when right-clicked
- **Hide Bow Pull Animation** - Hide the bow pulling animation
- **Gold Ingot Overlay** - Customize the display size of gold ingots on the ground
  - **Only Work in MurderMystery** - Only enable this feature in Murder Mystery mode
  - **Gold Ingot Overlay Multiplier** - Adjust the size multiplier for gold ingot overlay

### HUD

- **Show Secret Passage Status** - Display whether secret passages are open in certain maps
  - **Secret Passage Status HUD Settings** - Customize the secret passage status HUD
- **Show Weapon Cooldown** - Show cooldown HUD for throwing knives and bow
  - **Weapon Cooldown HUD Settings** - Customize the weapon cooldown HUD
- **Show Title Tips** - Display additional title tips for Murder Mystery
  - **Title Tips Language** - Select language for title tips (English/Chinese)
  - **Bow Picked Title** - Show title when bow is picked up
  - **Last Survivor Title** - Show title when you are the last survivor in infection mode
  - **Map Enabled Title** - Show title when map is enabled
  - **Title Tips Settings** - Customize the title tips HUD

### Sound

- **Enable Sounds** - Enable additional sound effects for Murder Mystery
- **Bow Dropped Sound** - Select sound effect when bow is dropped (None, Pling, Orb, Level Up, Meow, Harp, Anvil, Villager)
- **Bow Picked Sound** - Select sound effect when bow is picked up
- **Last Survivor Sound** - Select sound effect when last survivor appears
- **Map Enabled Sound** - Select sound effect when map is enabled
- **Alpha Appears Sound** - Select sound effect when alpha appears

这是一个我的世界1.8.9Forge 客户端Mod，是我在AI的帮助下开发的，能让你在游玩Hypixel MurderMystery时获得更好的体验，当前支持hypixel的简体中文和英文语言

注意:

使用此mod需要自负风险

## 功能介绍

### 通用

- **隐藏无击杀提示标题** - 隐藏杀手在1分钟内未获得击杀的标题提示
- **在Tab列表显示玩家身份** - 在Tab列表中显示已知玩家的身份
- **隐藏无用的盔甲架标签** - 隐藏地图中大部分隐形盔甲架的名称标签
- **隐藏粒子效果** - 隐藏游戏中的大部分粒子效果以提升性能
- **切换粒子显示** - 按键绑定（默认：H）来切换粒子显示
- **显示弓掉落粒子** - 在掉落的弓的位置显示粒子效果

### 聊天

- **隐藏Kali刷屏消息** - 隐藏聊天中的Kali刷屏消息
- **自动切换聊天频道** - 加入或离开组队时自动切换聊天频道
- **自动发送弓击杀距离** - 自动发送弓远距离击杀信息
  - **语言** - 选择距离消息的语言（英语/中文）
  - **发送到公开/组队/公会** - 选择在哪里发送距离消息
- **自动发送身份到组队** - 自动在组队中发送自己的身份信息
  - **语言** - 选择身份消息的语言（英语/中文）
- **组队指令快捷按钮** - 显示组队管理的快捷指令按钮

### 物品

- **禁用动作盔甲架** - 禁用动作盔甲架（可在shift状态下右键使用）
- **保护侦探弓** - 防止侦探弓被献祭给Kali（按住shift时仍可献祭）
- **保护匕首** - 防止一些匕首（例如船）因原版特性在右键时消失
- **隐藏拉弓动画** - 隐藏拉弓的动画
- **金锭覆盖层** - 自定义显示掉落地面的金锭大小
  - **仅在密室杀手中生效** - 仅在密室杀手模式中启用此功能
  - **金锭覆盖层倍率** - 调整金锭覆盖层的大小倍率

### HUD

- **显示秘密通道状态** - 在部分地图中显示秘密通道是否开启
  - **秘密通道状态HUD设置** - 自定义秘密通道状态HUD
- **显示武器冷却** - 显示飞刀和弓的冷却时间HUD
  - **武器冷却HUD设置** - 自定义武器冷却HUD
- **显示标题提示** - 为密室杀手添加更多的标题提示
  - **标题提示语言** - 选择标题提示的语言（英语/中文）
  - **弓被捡起标题** - 弓被捡起时显示标题
  - **最后幸存者标题** - 在感染模式中成为最后幸存者时显示标题
  - **地图启用标题** - 地图启用时显示标题
  - **标题提示设置** - 自定义标题提示HUD

### 音效

- **启用音效** - 为密室杀手添加更多的音效提示
- **弓掉落音效** - 选择弓掉落时的音效（无、叮、球、升级、喵、竖琴、铁砧、村民）
- **弓被捡起音效** - 选择弓被捡起时的音效
- **最后幸存者音效** - 选择最后幸存者出现时的音效
- **地图启用音效** - 选择地图启用时的音效
- **母体出现音效** - 选择母体暴露时的音效
